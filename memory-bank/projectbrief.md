# Project brief — Codryn Portfolio

## Goal
Public marketing / portfolio website for Codryn, a software development studio.

## Scope
- Single-page portfolio with Hero, About, Projects, Services, Contact, Footer
- Dark-mode-first developer-tool aesthetic
- Easy content edits via `src/data/*`
- Deployable to Vercel or Netlify

## Non-goals (for now)
- CMS / admin panel
- Authenticated areas
- Blog
